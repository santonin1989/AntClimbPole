# AntClimbPole

该项目基于 canvas 动画可视化了“蚂蚁爬杆”这一算法题目的过程。

具体问题情境如下：

- 有一根 300 厘米的细木杆，在第 30cm、80cm、110cm、160cm、250cm 这五个位置上各有一只蚂蚁。木杆很细，不能同时通过两只蚂蚁。

- 开始时，蚂蚁的头朝左还是朝右是任意的，它们只会朝前走或掉头，但不会后退。

- 当任意两只蚂蚁碰头时，两只蚂蚁会同时调头朝相反方向走。假设蚂蚁们每秒钟可以走 5cm 的距离。

- 不允许穿越对方身体继续执行。
